# Transport Tycoon Remake (TTM) - Game Design Document

## Overview
A complete transport empire simulation game built as a React web app. Players build and manage transportation networks to connect towns, industries, and generate profit.

## Core Gameplay Loop
1. **Build Infrastructure** - Lay roads, rails, build stations, airports, docks
2. **Purchase Vehicles** - Trains, trucks, buses, ships, planes
3. **Create Routes** - Assign vehicles to connect pickup and delivery points
4. **Earn Money** - Transport passengers and cargo between locations
5. **Expand** - Reinvest profits into larger networks
6. **Win** - Reach target net worth or dominate all transport

## Map System
- **Size**: 1024x1024 tiles (viewable via scrolling camera)
- **Terrain Types**: Grassland, Water, Hills, Mountain, Desert, Snow
- **Procedural Generation**: Perlin noise-based terrain with rivers and lakes
- **Features**: Trees, houses that grow into towns/cities

## Towns & Cities
- Start with 4-6 small towns
- Towns grow based on passenger/cargo service quality
- Growth stages: Hamlet (50) → Village (100) → Town (250) → City (500) → Metropolis (1000)
- Each town generates passengers and requests cargo

## Industries (12 types)
1. Coal Mine → produces Coal
2. Iron Ore Mine → produces Iron Ore
3. Quarry → produces Stone
4. Lumber Mill → produces Wood
5. Farm → produces Food/Milk
6. Oil Well → produces Oil
7. Steelworks → consumes Iron Ore, produces Steel
8. Paper Mill → consumes Wood, produces Paper
9. Bakery → consumes Food, produces Baked Goods
10. Car Factory → consumes Steel, produces Cars
11. Spaceport → consumes Cars, produces Space Equipment
12. Refinery → consumes Oil, produces Fuel

## Construction System
### Ground Infrastructure
- **Roads**: Basic connections, cheap, for buses/trucks
- **Railways**: High capacity, expensive, for trains
- **Signals**: For two-way train traffic
- **Tunnels**: Under hills/mountains
- **Bridges**: Over water and terrain

### Stations & Terminals
- **Train Station**: For trains (8 tiles long)
- **Bus Stop**: For buses (2 tiles)
- **Truck Stop**: For trucks (2 tiles)
- **Airport**: For planes (11x11 tiles)
- **Dock**: For ships (water adjacent, 3 tiles)

## Vehicles (5 types, multiple variants each)

### Trains (6 variants)
- Steam Locomotive (slow, cheap)
- Diesel Locomotive (medium)
- Electric Locomotive (fast)
- High Speed Train (very fast, expensive)
- Cargo Wagon variants
- Passenger Car variants

### Road Vehicles (8 variants)
- Truck (various cargo types)
- Bus (passenger only)
- Minibus, Coach, Cargo Truck, etc.

### Aircraft (4 variants)
- Propeller plane (short range)
- Jet aircraft (long range)
- Cargo plane variants

### Ships (4 variants)
- Cargo ship, Passenger ship, Ferry

### Each vehicle has:
- Speed, capacity, reliability, cost
- Maintenance cost per month
- Can be purchased, scrapped, or stored

## Economy System
- **Starting Cash**: $100,000
- **Bank Loans**: Up to $4,000,000 in $10,000 increments
- **Interest Rate**: Configurable (default 5%)
- **Income**: Fare × distance × passengers/cargo delivered
- **Expenses**: Vehicle purchase, maintenance, construction, loan interest
- **Inflation**: Prices rise over time (1% per year)

## Cargo System
- **Passengers**: Generated by towns, want to go to other towns or work
- **Mail**: Between towns
- **Goods**: Between industries (coal, iron, steel, wood, food, oil, etc.)
- Each cargo type has a value and demand

## Game Controls
- **Mouse**: Click to select, drag to build, right-click to cancel
- **Arrow Keys / WASD**: Pan camera
- **Scroll Wheel**: Zoom in/out
- **Number Keys 1-9**: Select construction tool
- **Space**: Pause/unpause
- **Esc**: Close panels

## UI Panels
1. **Construction Toolbar** (bottom): Tool icons for building
2. **Vehicle Panel**: Buy, manage, assign vehicles
3. **Economy Panel**: Money, loans, profit/loss graphs
4. **Map Panel**: Overview mini-map
5. **Info Panel**: Click any object to see details
6. **Settings Panel**: Game speed, sound, difficulty

## Graphics Style
- **Isometric-ish top-down view** with colorful pixel art style
- Canvas-based rendering for performance
- Smooth animations for vehicles
- Color-coded terrain
- Animated water, growing towns
- Clear UI with icons

## Sound
- Engine sounds per vehicle type
- Construction sounds
- UI click/selection sounds
- Background music (procedural/generated)

## Save/Load
- Full game state serialization
- Auto-save every 5 in-game months
- Manual save/load via localStorage
- Export/import as JSON file

## Game Settings
- **Date Range**: 1920-2020 (100 year game)
- **Difficulty**: Easy, Normal, Hard (affects starting money, competition)
- **Building Limits**: Configurable
- **Construction Cost**: Multiplier
- **Vehicle Cost**: Multiplier

## Win/Lose Conditions
- **Win**: Reach target net worth by end date
- **Lose**: Bankruptcy (negative money + no loans available)
- **Target**: Varies by difficulty ($2M easy, $4M normal, $8M hard)

## Technical Architecture
- React + Vite for fast development
- Canvas API for map rendering
- useReducer for game state
- requestAnimationFrame for game loop
- localStorage + file download for saves
- Web Audio API for sounds
